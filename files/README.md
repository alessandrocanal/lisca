#Rails app based on Lisca startup template

https://bitbucket.org/batdevis/lisca

##Features

###Main Gems
* _swagger_engine_: api documentation
* _capistrano_: deploy
* _rspec_: test
* _figaro_: config vars
* _redis-rails_: cache store

###System Requirements
* _ruby 2.2_

###Backing Services
* _postgresql 9.4_
* _redis_
